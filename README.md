## Про проект

Личный кабинет для подключения и просмотра отзывов из Яндекс Карт. Фронт — Vue 3 (Inertia), бек — Laravel 12 с очередью парсинга.


## Полезные скрипты
- Проверка статики: `composer phpstan`
- Проверка стиля: `composer cscheck`
- Автоисправление стиля: `composer csfix`

## Заметки
- Конфиг PHPStan: `phpstan.neon.dist`
- Конфиг PHP-CS-Fixer: `.php-cs-fixer.dist.php`
